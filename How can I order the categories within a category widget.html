# How can I order the categories within a category widget? Learn how to order category widgets based on an aggregation operation in CARTO Builder. Builder users can order the 5 top categories shown in a category widget from the widget options. In addition to `COUNT`, they can be ordered based on an agregation from another numeric field such as `SUM`, `AVG`, `MIN` or `MAX` as shown in this animated gif: ![order](/help/images/building-maps/order-categories/order.gif)
